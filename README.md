### Сборка и отладка

Локальная сборка проекта:
```
yarn start
// or
npm run start
```

Выпуск новой версии:
- перед публикацией не забудте изменить версию в `package.json`
```
yarn build
// or
npm run build
```

Сборка демо версии
```
yarn demo
// or
npm run demo
```
